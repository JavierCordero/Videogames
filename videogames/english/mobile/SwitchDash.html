Switch Dash (clone)

Just like the games found in the PS4 part, in this case we have a cross-platform clone (for Android and PC) of the popular game Switch Dash

This game is made in Java using Android Studio as an IDE.

In it, we have the classic series of menus within a game for mobile devices and their typical life cycle. The game consists of matching the lower bar that represents the player with the color of the balls that are falling, increasing their speed every certain number of correct balls.
